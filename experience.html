My Journey as a Beginner Web Developer

Past: The Beginnings of My Web Development Journey

My journey into web development began with a fascination for how websites function and a desire to create something of my own. I started with online tutorials and courses, immersing myself in the basics of HTML and CSS. These initial steps were exciting and challenging, as I learned how to structure a webpage and style it to look appealing.

Present: Growing as a Web Developer

    One of the key aspects of my present journey in web development is enhancing my core skills in HTML, CSS, and JavaScript through Treehouse. This learning platform has provided me with structured courses and interactive content that have been instrumental in my growth.

Future: Aspiring to New Heights

Looking ahead, my goal is to become a full-stack developer, proficient in both front-end and back-end technologies. To achieve this, I plan to deepen my knowledge of modern programming languages and frameworks, allowing me to build robust, scalable applications that handle all aspects of web development seamlessly.
